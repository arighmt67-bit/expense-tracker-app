# Expense Tracker

A simple expense tracker web app to manage personal finances. Built as a portfolio project using vanilla HTML, CSS, and JavaScript — no frameworks or dependencies.

## Features
- Add and delete income & expense transactions
- Real-time balance, total income, and total expense summary
- Data persisted using localStorage
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
